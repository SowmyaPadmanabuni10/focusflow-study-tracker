# FocusFlow - Smart Study Productivity Tracker

FocusFlow is a Python-based application developed to help users track study sessions and productivity levels.

## Features
- Add study records
- View study sessions
- Calculate total study hours
- Average focus score analysis
- Most studied subject tracking
- Daily study goal checking
- Save records to text file

## Technologies Used
- Python
- Lists
- Dictionaries
- File Handling

## Concepts Practiced
- Loops
- Conditional Statements
- Data Structures
- User Input Handling
- File Operations
